清远市人社局组织召开留学人员座谈会

发布时间：2016-10-09 16:33:40 发布人：综合规划和政策法规科 阅读次数：182

为进一步落实人才强市战略，加强留学人员回国服务体系建设，加大对留学人员来清创业的支持力度，推动我市经济社会发展，10月8日下午，市人社局在市政府机关2号楼组织召开了留学人员座谈会。会议主要通报了我市留学人员服务工作情况，围绕服务留学人员在创新创业方面问题进行座谈和讨论。副市长陈建华，市政府副秘书长郭蔚丽，市人社局局长梁思，相关部门负责人及留学人员代表约20人参加了会议。会议由市政府副秘书长郭蔚丽主持。

会上，梁思通报了我市留学人员服务工作的总体情况。她指出，我市高度重视留学人员服务工作，出台了相关支持留学归国人员创业配套政策，成立了清远市归国留学人员创业园，打造适合留学人才创新创业的产业发展基地，搭建创新创业人才服务的综合性平台，为留学回国人员来清创业营造良好的环境和提供创业条件。深入调研，密切联系，积极宣传创新创业政策，加大力度引进优秀留学人员。同时，帮扶引导，做好留学人员回国工作资助项目。广泛宣传留学人员回国工作资助项目，及时鼓励和帮扶符合条件的留学人员申报。下一步，我们将进一步加快完善留学人员创业优惠政策，成立清远市外国专家和留学人员服务中心，搭建多功能服务平台，进一步加快建设特色归国留学人员创业园。
