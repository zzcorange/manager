市人社局领导到广东岭南职业技术学院开展实训中心建设工作调研

| 2016-10-09 16:33:40 | 举报 阅读次数：182

为加强我市高技能人才工作，进一步加快我市高技能人才实训基地建设，6月15日，实训中心筹建工作人员在市人社局局长梁思的带领下前往广东岭南职业技术学院清远校区调研，市人社局副局长彭德才等参加了本次调研。

调研组一行在岭南教育集团执行董事钟燕锋的陪同下参观了学院的商业体验和服务实训中心、现代物流实训中心、药物制剂技术实训中心及机电技术实训中心等实训场（室），同时围绕实训中心的建设功能、布局问题进行了具体的探讨。

高技能实训中心是政府向社会提供高技能实训和鉴定服务的公共服务平台。我市去年起启动了高技能人才公共实训基地建设工作，目前建设进入功能规划和初步设计阶段，为更好地体现我市实训特色，打造未来实训品牌，局领导十分重视，成立了以梁思局长为组长的筹建领导小组，并多次组队到香港、东莞、广州、惠州等地实地考察和调研。

训基地的建设是我市贯彻落实《中共中央办公厅国务院办公厅关于进一步加强高技能人才工作的意见》一项重要举措，市人社局发挥在高技能人才工作中的公共管理和服务职能，通过搭建高层次公共实训平台，解决高技能人才培养评价服务手段不足和水平不高等问题，带动高技能人才各项工作，加快本地区技能人才队伍建设。
